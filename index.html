PDF Bookmark Splitter

Upload a bookmarked PDF, detect bookmark start pages, preview each segment, and export all segments as separate PDFs in a ZIP using a custom filename pattern.

Settings

This runs fully in your browser. Your PDF is not uploaded anywhere by this page.
Most PDFs use top-level bookmarks for major sections.
Available placeholders: {orig}, {bookmark}, {page}, {index}, {start}, {end}, {pages}, {level1}, {level2}, {level3}, {level4}, {level5}, {level6}.
Suggested: {orig}_p{page}_{bookmark}
Range: {orig}_{start}-{end}_{bookmark}
Indexed: {orig}_{index}_{bookmark}
Hierarchy: {level1}_{level2}_{bookmark}
Deep: {level1}_{level2}_{level3}_p{page}
Clean legal: {orig} - {level1} - {level2} - p{page}
Bookmark first: {bookmark} - {orig} - p{page}
Full path: {level1}_{level2}_{level3}_{level4}
Page range path: {level1}_{level2}_{start}-{end}
Index + path: {index}_{level1}_{level2}_{bookmark}
Load a PDF, then click “Analyze bookmarks”.
Best results: open this page through http://localhost instead of directly as a file:// HTML file. Browser security can block PDF worker loading and ZIP export on local file URLs. Notes: this app splits at the bookmark destinations it can resolve. Some PDFs have bookmarks pointing to named destinations or odd internal objects; those are handled where possible, but a few unusual files may need adjustment in Acrobat first.
